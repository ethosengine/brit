# gix-archive

Generates archives (tar, tar.gz, zip) from a worktree stream, similar to `git archive`.

**Tier:** plumbing
**Errors:** gix-error
**Notable:** Early implementation; tar support is basic and zip symlinks may not restore on macOS.

See root `CLAUDE.md` for branch discipline, agent roster (gix-architect / gix-steward / gix-runner), and the parity-loop structure.
