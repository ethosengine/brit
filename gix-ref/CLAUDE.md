# gix-ref

Handle git references (loose files, packed-refs, symbolic and peeled targets).

**Tier:** plumbing
**Errors:** thiserror
**Notable:** File-based ref store; lock files acquired via gix-lock.

See root `CLAUDE.md` for branch discipline, agent roster (gix-architect / gix-steward / gix-runner), and the parity-loop structure.
