# gix-mailmap

Parse `.mailmap` files and resolve signatures via an accelerated Snapshot.

**Tier:** plumbing
**Errors:** gix-error
**Notable:** Lookup is case-insensitive by email (optionally name).

See root `CLAUDE.md` for branch discipline, agent roster (gix-architect / gix-steward / gix-runner), and the parity-loop structure.
