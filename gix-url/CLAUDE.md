# gix-url

Parses and serializes git URLs (ssh, git, http(s), file, scp-style) with security-aware helpers.

**Tier:** plumbing
**Errors:** thiserror
**Notable:** Browser-grade URL parser; may be more restrictive than git (per SHORTCOMINGS.md).

See root `CLAUDE.md` for branch discipline, agent roster (gix-architect / gix-steward / gix-runner), and the parity-loop structure.
