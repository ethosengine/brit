# gix-prompt

Git-style terminal prompting with `GIT_ASKPASS` and `askpass` program support.

**Tier:** plumbing
**Errors:** thiserror
**Notable:** Unix-only; returns `UnsupportedPlatform` error elsewhere.

See root `CLAUDE.md` for branch discipline, agent roster (gix-architect / gix-steward / gix-runner), and the parity-loop structure.
